# Tech Context

## Core Technologies
-   **Python**: 3.9+
-   **Streamlit**: Web framework for data apps.
-   **Pandas**: Data manipulation and analysis.
-   **Requests**: HTTP library for API calls.
-   **Playwright**: Browser automation for scraping.

## External Services
-   **KOBIS API**: Korean Box Office Information System.
    -   Endpoint: `http://www.kobis.or.kr/kobisopenapi/webservice/rest/boxoffice/searchDailyBoxOfficeList.json`
    -   Auth: API Key required.
-   **Naver News**: Target for web scraping (No API used here).

## Development Setup
-   Virtual Environment: Recommended (`venv`).
-   Package Management: `pip`.
